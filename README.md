# KDE Icon Helper

KDE Icon Helper is a small desktop utility for Linux that helps inspect and repair launcher icons.

It is mainly useful for `.desktop` launchers and Windows app shortcuts that were added to the desktop through tools like Wine, Lutris, or similar setups.

## Features

1. Scan desktop items and launchers
2. Detect missing or broken icons
3. Preview the currently resolved icon
4. Fix launcher icons automatically
5. Set a custom icon manually
6. Restore the default icon from backup
7. Navigate entries quickly with the keyboard

## What it is for

KDE Icon Helper makes it easier to clean up launcher icons when desktop entries point to broken files, missing icons, or manually overridden paths.

It focuses on a simple workflow:

1. Scan items
2. Inspect the selected launcher
3. Fix or replace the icon
4. Restore the original state when needed

## Current scope

The app is designed for Linux desktop environments and works best with `.desktop` launchers.

Some launcher types may behave differently depending on how they were created.
